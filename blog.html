Question-1: What are the different ways to select an element in the DOM?

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

In the DOM, elements can be selected using methods such as getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll, and getElementsByName. These methods provide various ways to access and manipulate elements based on their ID, class, tag, CSS selectors, or name attribute.

Question-2: What is the difference between innerHTML, innerText, and textContent?

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

innerHTML allows for reading or setting HTML content, including tags and nested elements, within an element. innerText provides the visible text content, excluding hidden elements, while accounting for CSS styling. textContent retrieves or sets all text content within an element, including hidden elements, but ignores HTML tags and styling.

Question-3: What is event delegation in the DOM?

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Event delegation in the DOM is a technique where a single event listener is added to a parent element instead of multiple child elements. When an event occurs on a child, it bubbles up to the parent, which handles the event. This improves efficiency and simplifies code management.

Question-4: What is event bubbling in the DOM?

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Event bubbling in the DOM is a mechanism where an event starts from the target element and propagates upward through its ancestor elements. This means that event handlers for the target element execute first, followed by its parents, grandparents, and so on, allowing multiple elements to respond to a single event.

Question-5: How do you create, add, and remove elements using JavaScript?

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

In JavaScript, you create elements using document.createElement(), add them with methods like appendChild() or insertBefore(), and remove them using removeChild() or remove(). For example, let div = document.createElement('div'); parent.appendChild(div); parent.removeChild(div); demonstrates creating, adding, and removing a div element.
